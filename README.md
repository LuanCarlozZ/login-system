# 🔐 Sistema de Login em Python

Projeto simples de autenticação feito em Python.

## Funcionalidades

* Menu interativo
* Login com usuário e senha
* Limite de 3 tentativas
* Bloqueio após erros

## Como executar

1. Abra o terminal
2. Execute:

```bash
python login.py
```
